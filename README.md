# Frontend
## Client app
### Step 1
Install node.js
### Step 2
Open command line

Run command `npm install -g @angular/cli`
### Step 4
Change to root folder
### Step 5

### Step 6
Run command `npm start`
### Step 7
Open the browser

Access the url `https://localhost:4200`
